## 一、项目概述

### 1.1 项目名称
**庐山文旅通** - 庐山文化旅游信息服务平台

### 1.2 项目定位
基于庐山丰富的历史文化、自然风光资源，打造一套集信息查询、导览服务、文化展示、行程规划于一体的综合性旅游服务平台。

### 1.3 目标用户
- 前往庐山旅游的游客
- 对庐山文化感兴趣的研究者
- 旅行爱好者、摄影爱好者
- 学生及教育工作者

## 二、当前系统状态（2026-03-24）

### 2.1 技术栈
- **前端框架**: Streamlit
- **后端语言**: Python 3.12
- **数据格式**: YAML
- **地图库**: Folium
- **天气 API**: Open-Meteo（免费）

## 五、数据结构设计

### 5.1 景点表 (scenic_spots)
```yaml
- id: 主键
- name: 景点名称
- category: 分类（自然景观/人文景观/宗教建筑）
- subcategory: 子分类
- altitude: 海拔高度
- description: 详细介绍
- location: 地理位置（经纬度）
- tags: 标签数组
- image: 图片 URL
- visiting_time: 建议游览时间
- difficulty: 难度等级
- opening_hours: 开放时间 ✅新增
- ticket_price: 门票价格 ✅新增
- best_season: 最佳游览季节 ✅新增
```

### 5.2 诗词表 (poems)
```yaml
- id: 主键
- title: 诗词标题
- author: 作者
- author_id: 作者 ID
- dynasty: 朝代
- content: 诗词内容
- location: 创作地点
- location_id: 关联景点 ID
- background: 创作背景
- type: 类型（写景/抒情/哲理）
```

### 5.3 美食表 (restaurants) ✅新增
```yaml
- id: 主键
- name: 餐厅名称
- category: 菜品分类
- price_range: 价格区间
- avg_price: 人均消费
- address: 地址
- description: 介绍
- signature_dishes: 招牌菜
- rating: 评分
- location: 地理位置
- opening_hours: 营业时间
```

### 5.4 住宿表 (hotels) ✅新增
```yaml
- id: 主键
- name: 酒店名称
- category: 住宿类型
- price_range: 价格区间
- avg_price: 每晚价格
- address: 地址
- description: 介绍
- amenities: 设施列表
- rating: 评分
- location: 地理位置
- phone: 联系电话
```

## 七、技术债务与优化建议

### 7.1 当前问题
1. 图片使用 Unsplash 随机图，非真实庐山照片
2. AI 导游使用预设回答，未接入真实大模型
3. 用户数据（足迹、收藏）存储在 Session，刷新后丢失
4. 部分页面加载速度较慢

### 7.2 优化建议
1. 收集或使用官方授权的真实庐山图片
2. 接入 Claude API 或其他大模型
3. 引入 SQLite 数据库持久化用户数据
4. 使用 `@st.cache_data` 优化大数据加载

## 八、部署方案

### 8.1 本地运行
```bash
pip install -r requirements.txt
streamlit run app.py --server.port 8503
```

### 8.2 云端部署（推荐）
- **Streamlit Cloud**: 免费托管，连接 GitHub 即可
- **Docker**: 容器化部署
- **云服务器**: 使用 systemd 后台运行

### 8.3 生产环境配置
- 启用 HTTPS
- 配置域名
- 添加访问统计
- 设置错误日志
